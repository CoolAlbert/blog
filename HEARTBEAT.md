# Heartbeat Checklist

This file controls heartbeat and agent message handling.

## Regular Heartbeat Tasks

Rotate through these checks 2-4 times per day:

- **THOUGHTS.md** - Check if there are thoughts to record
- **BLOG_IDEAS.md** - Check if there are blog ideas to capture or post ideas to write about
- **Calendar** - Check upcoming events in next 24-48h
- **Weather** - Check if relevant for upcoming activities

Only reach out if:
- Important calendar event coming up (<2h)
- Something urgent requires attention
- It's been >8h since you said anything
- There are thoughts worth recording (add to THOUGHTS.md silently)
- There are blog ideas (check BLOG_IDEAS.md, remind if there are ideas ready to write)

**Stay quiet during:** 23:00-08:00 (Moscow time) unless urgent.
